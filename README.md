# Five Senses Educational Platform (منصة الحواس الخمس)

**Five Senses** is an AI-powered, educational platform designed to transform how students interact with their learning materials. The core philosophy of the platform is built around the idea that linking different senses during the study process leads to a better, more retentive learning experience. 


## The Core Concept

The fundamental idea behind Five Senses is to move beyond passive reading. By converting traditional study materials into multi-sensory, interactive experiences, learners can engage with content through sight, sound, and interactive tasks, maximizing their cognitive absorption and retention.

To achieve this, the platform provides a seamless, multi-modal ingestion system. Students can bring in knowledge from anywhere to be transformed:

- **Document Uploads**: Drag and drop PDF study guides, presentations, or worksheets.
- **Web Content**: Paste URLs to extract and study from educational articles or web pages.
- **Raw Text**: Quickly paste notes or text snippets directly into the platform.

Once materials are ingested, the platform utilizes AI to break down and translate this content into tailored, multi-sensory study sessions. The **Review Center** (Dashboard) provides an interactive space to track recent study sessions, monitor learning progress, and stay on top of educational goals.

## Getting Started

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation & Setup

1. **Navigate to the project directory**:
   ```bash
   cd "Five Senses"
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```

4. **View the platform**:
   Open [http://localhost:3000](http://localhost:3000) in your browser.
